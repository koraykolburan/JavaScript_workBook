JavaScript Data Types

JavaScript use this: = + - * / % ++ -- .

  1. Number.
  2. String: using double "" and single '' quotes delimiter. It's a escape character: \ .
  3. Boolean: True or False. &&(and), ||(or), !(not).

    true && true = true
    true && false = false
    false && true = false
    false && false = false

    true || true = true
    true || false = true
    false || true = true
    false || false = false

    !false = true
    !true = true
  4. Null.
  5. Bigint.
  6. Undefined.
  7. Symbol.
  8. Object.
