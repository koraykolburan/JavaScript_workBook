JavaScript Data Types

JavaScript use this: = + - * / % ++ -- .

  1. Number.
  2. String: using double "" and single '' quotes delimiter. It's a escape character: \ .
  3. Boolean: True or False. &&(and), ||(or), !(not).

    true && true = true
    true && false = false
    false && true = false
    false && false = false

    true || true = true
    true || false = true
    false || true = true
    false || false = false

    !false = true
    !true = true
  4. Null.
  5. Bigint.
  6. Undefined.
  7. Symbol.
  8. Object.
  • You can learn which data type you are using with this: "typeof".

JavaScript Functions

For machines: "Block of code designed to perform a particular task."

Some data to input.
Performs a task.
Returns a result.

Example:
function myFunction() {
      statements...
      return...
}

All the code in the body of the function is NOT executed, until we "call" the function.
