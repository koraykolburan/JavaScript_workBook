JavaScript Data Types

JavaScript use these operators: = + - * / % ++ -- .

  1. Number.
  2. String: using double "" and single '' quotes delimiter. It's a escape character: \ .
  3. Boolean: True or False. &&(and), ||(or), !(not).

    true && true = true
    true && false = false
    false && true = false
    false && false = false

    true || true = true
    true || false = true
    false || true = true
    false || false = false

    !false = true
    !true = true
  4. Null.
  5. Bigint.
  6. Undefined.
  7. Symbol.
  8. Object.
  • You can learn which data type you are using with this: "typeof".

JavaScript Variables: var, let, const.

A Variable is a name or a tag of that references spacing a memory where to store information.

  1. Always declare variables.
  2. Always use const if the value should not be change.
  3. Always use const if the type should not be change.
  4. Always use let if you can't use const.
  5. Always use var if you MUST support old browsers.

JavaScript Functions

A function is a name or a tag that references spacing a memory where we store code.

For machines: "Block of code designed to perform a particular task."

Some data to input.
Performs a task.
Returns a result.

Example:
function myFunction() {
      statements...
      return...
}

All the code in the body of the function is NOT executed, until we "call" the function.

Functions = Verb

JavaScript Scopes

  1. Function Scope: That variable exists inside of the function scope.
  2. Global Scope: The variable accessible within in our JavaScript code.
  3. Leaking Variable: If you create a variable without using a var statement or another words within the function, JavaScript runtime will take care of declaring the variable for you and then assignment will be executed.
